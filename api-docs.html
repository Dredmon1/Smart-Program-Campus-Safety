GET /api/incidents List incidents (filterable)
Query Parameters
Name Type Description
status string Filter: Active, Responding, Cleared, Closed
severity string Filter: Low, Medium, High, Critical
region string Filter: Los Angeles, Orange County, Inland Empire, San Diego
Response (200)
{
  "count": 4,
  "incidents": [
    {
      "id": 1,
      "title": "Cargo Theft — Port of LA",

      "severity": "High",
      "status": "Active",

      "region": "Los Angeles",
      "timestamp": "2026-02-19T12:30:00"
    }
  ]
}
POST /api/incidents Create new incident
Request Body (JSON)
Name Type Description
title string Required — Incident title
type string Required — Theft, Traffic, Fire, etc.
severity string Required — Low, Medium, High, Critical
location object {lat, lng} — defaults to LA center
assignedUnit string Unit ID (e.g. UNIT-12)
Response (201)
{
  "message": "Incident created",
  "incident": { ... }
}
